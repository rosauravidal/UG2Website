Development Kit

The Development Kit consists of a Docker file containing the basic structure for the algorithms submission, as well as the quantitative classification code for the second challenge. The images, annotation, and lists specifying the training/validation sets for the challenge are provided separately, and can be obtained via: Google Drive.

Installation and Configuration

The Development Kit will be available as a Docker image at Docker Hub.
